# Online Judge Project

## Overview
An Online Judge platform where users can solve coding problems, submit solutions, and get evaluated automatically.

## Features
- User authentication using JWT
- Problem CRUD APIs
- Submission CRUD APIs
- Test case management
- Secure backend APIs

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- JWT Authentication
